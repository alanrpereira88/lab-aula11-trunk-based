# Sobre o Projeto

Este repositório é usado na disciplina **DevOps I** do curso de Bacharelado em Ciência da Computação da Univértix.

## Objetivo

Praticar o **Trunk-Based Development**: contribuir diretamente na `main` com branches de vida curtíssima, refletindo o ritmo de times que praticam entrega contínua.

## Regra do lab

Você tem **20 minutos** para criar a branch, adicionar seu perfil e integrar na `main`. Se não terminar, simplifique — não atrase o merge.
